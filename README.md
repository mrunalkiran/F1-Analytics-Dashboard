# 🏎️ F1 Analytics Dashboard

This project analyzes Formula 1 race data using Python and Tableau.

## 🔧 Tech Stack
- Python (Pandas)
- Tableau Public
- GitHub

## 📊 Dashboard Insights
- Driver performance over time
- Total wins comparison
- Race strategy analysis (grid vs position)

## 📁 Project Structure
- data/ → raw datasets
- notebook/ → data processing
- tableau/ → final dashboard

## 🚀 Outcome
Built an interactive dashboard to analyze driver performance trends and race strategies.
